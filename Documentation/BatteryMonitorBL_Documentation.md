# Документация по приложению BatteryMonitorBL

## Общее описание
BatteryMonitorBL - это iOS-приложение для мониторинга и управления батареями через Bluetooth-соединение. Приложение позволяет пользователям подключаться к батарейным модулям, отслеживать их состояние в реальном времени, просматривать детальную информацию о батарее и настраивать параметры модуля.

## Структура приложения
Приложение состоит из 4 основных экранов:

### 1. Главный экран (HomeViewController)
- Отображает основную информацию о батарее:
  - Уровень заряда батареи (в процентах и визуально)
  - Статус батареи (зарядка, разрядка, защита, ожидание)
  - Напряжение (Voltage)
  - Ток (Current)
  - Температура (Temperature) в градусах Цельсия и Фаренгейта
  - Время последнего обновления данных
- Имеет кнопку для перехода на экран подключения устройств

### 2. Экран детальной информации (DetailsViewController)
- Отображает подробную информацию о батарее:
  - Напряжение каждой ячейки батареи (Cell Voltage)
  - Температура PCB (печатной платы)
  - Температура каждой ячейки батареи (Cell Temperature)
- Данные представлены в виде коллекции с двумя секциями:
  - Секция напряжения ячеек
  - Секция температуры PCB и ячеек

### 3. Экран подключения (ConnectivityViewController)
- Позволяет пользователю управлять Bluetooth-соединением:
  - Сканировать доступные устройства
  - Подключаться к выбранному устройству
  - Отключаться от текущего подключенного устройства
- Имеет два режима отображения:
  - Режим "не подключено": показывает только список доступных устройств
  - Режим "подключено": показывает подключенное устройство и список доступных устройств

### 4. Экран настроек (SettingsViewController)
- Отображает и позволяет изменять настройки устройства:
  - Версия приложения (только для просмотра)
  - Идентификатор модуля (Module ID) - от 1 до 16
  - Протокол CAN (доступен только если Module ID = 1)
  - Протокол RS485 (доступен только если Module ID = 1)
- При входе на экран настроек приостанавливается обновление данных BMS, при выходе - возобновляется

## Основные компоненты приложения

### ZetaraManager
Центральный компонент, отвечающий за взаимодействие с устройством через Bluetooth:
- Сканирование доступных устройств
- Подключение и отключение от устройств
- Получение данных о состоянии батареи (BMS)
- Получение и установка настроек устройства (Module ID, RS485, CAN)
- Периодическое обновление данных о батарее

### Данные о батарее (Data.BMS)
Структура, содержащая информацию о состоянии батареи:
- Напряжение (voltage) в вольтах
- Ток (current) в амперах
- Напряжение каждой ячейки (cellVoltages)
- Температура каждой ячейки (cellTemps)
- Температура PCB (tempPCB)
- Температура окружающей среды (tempEnv)
- Состояние заряда (soc) в процентах
- Состояние здоровья (soh) в процентах
- Статус батареи (status): зарядка, разрядка, защита, ожидание
- Количество ячеек (cellCount)

### Данные управления (ControlData)
Структуры для управления устройством:
- ModuleIdControlData: управление идентификатором модуля
- RS485ControlData: управление протоколом RS485
- CANControlData: управление протоколом CAN

### Визуальные компоненты
- BatteryView: отображает уровень заряда батареи визуально
- BatteryInfoView: отображает информацию о батарее (уровень заряда, статус)
- ComponentView: отображает компоненты на главном экране (напряжение, ток, температура)
- SettingItemView: отображает элементы настроек

## Принцип работы приложения
1. При запуске приложение проверяет, подключено ли устройство
2. Если устройство не подключено, пользователь может перейти на экран подключения и выбрать устройство
3. После подключения к устройству приложение начинает периодически получать данные о состоянии батареи
4. Пользователь может просматривать основную информацию на главном экране, детальную информацию на экране деталей и изменять настройки на экране настроек
5. Данные обновляются в реальном времени, показывая текущее состояние батареи

## Заключение
BatteryMonitorBL - это специализированное приложение для мониторинга батарей, которое предоставляет пользователям подробную информацию о состоянии батареи и позволяет настраивать параметры устройства. Приложение использует Bluetooth для связи с устройством и обновляет данные в реальном времени.
